Adding a Resource
To add a file:
1. Drop the file into resources/
2. Open resources.html and add a card in the relevant section:

<div class="resource-card">
  <div class="resource-icon">🖼</div>
  <div class="resource-info">
    <p class="resource-name">My Asset</p>
    <p class="resource-path">resources/my-asset.png</p>
  </div>
</div>

3. Commit both files to git normally.
